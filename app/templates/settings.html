🧠 Simulation Model Inputs

Every game runs 100,000 Monte Carlo simulations. Here's everything factored in:

⚾ Pitching
  • ✦ Season ERA & FIP
  • ✦ WHIP & K/9
  • ✦ Platoon splits (L/R)
  • ✦ Last 5 starts form
  • ✦ Pitch count fatigue
  • ✦ Carry-over (115+ pitches)
  • ✦ Days rest
  • ✦ IL status impact
🏏 Batting
  • ✦ Season AVG / OBP / SLG
  • ✦ Lineup slot PA expectations
  • ✦ vs LHP / RHP splits
  • ✦ Barrel rate (Statcast)
  • ✦ Hot / cold streaks (L7)
  • ✦ IL player impact score
  • ✦ Prop projections (H/HR/RBI/TB)
🏟️ Game Context
  • ✦ Home field advantage
  • ✦ Ballpark run factor
  • ✦ Umpire K-rate tendency
  • ✦ Catcher framing & arm
  • ✦ Bullpen depth & ERA
  • ✦ F3 / F5 / F7 inning models
🌤️ External Factors
  • ✦ Game-time temp & wind
  • ✦ Rain probability
  • ✦ HR wind boost %
  • ✦ Vegas ML / run line odds
  • ✦ Public betting % splits
  • ✦ Sharp money indicator
Run line (-1.5 / +1.5 / -2.5), O/U, and Same Game Parlay use the full correlated score distribution from all 100k simulated final scores.
